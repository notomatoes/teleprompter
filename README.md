# Teleprompter

一个简易网页版提词器，支持全屏自动滚动和中线高亮阅读。

## 功能

- 大字体提词显示
- 手动速度 / 按总时长自动速度
- 一键 `开始提词`（进入全屏并从顶部开始）
- 高亮开关、颜色、行数（1~3 行）
- 左右留白、上下留白可调
- 深色/浅色模式适配

## 本地使用

1. 直接在浏览器打开 `index.html`
2. 或在项目目录启动静态服务：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 控件说明

- `开始提词`：进入全屏并从头开始滚动
- `回到顶部`：停止并回到开头
- `显示/隐藏编辑区`：切换文本编辑面板
- `速度模式`：
  - `手动速度`：使用 `px/s`
  - `按总时长`：按输入时长自动计算滚动速度
